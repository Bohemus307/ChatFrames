## Getting Started

* Fork Repo to working directory and cd into ChatFrames
* Npm install
* .ENV file requires either dev or prod based on config.js

### Dependencies

* Node.js
* React
* React-dom
* Express.js

### Executing program

In development -
* npm run react-dev
* npm run server-dev
* Open localhost:3030
